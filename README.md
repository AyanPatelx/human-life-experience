# A Human Life

An interactive cinematic web experience about memory, growth, love, exhaustion, and reflection.

This project is not meant to feel like a traditional website.

It is designed as a slow emotional journey —
something between cinema, poetry, ambient music, and interactive art.

Scrolling through the experience changes:
- atmosphere
- color
- pacing
- motion
- emotional tone
- soundtrack

The website evolves like a human life.

---

## Experience

The journey moves through different emotional chapters:

1. Childhood  
2. Becoming Someone  
3. Love & Connection  
4. The Machine Years  
5. Reflection  

Each section blends visual atmosphere with dynamic ambient audio.

---

## Features

- Cinematic scroll experience
- Dynamic soundtrack transitions
- Scroll-reactive audio intensity
- Particle-based atmosphere system
- Ambient visual rendering
- Custom cursor interaction
- Emotional pacing design
- Responsive fullscreen experience

---

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- Dynamic Audio System

---

## Audio Design

The soundtrack changes depending on where the user is in the experience.

Instead of abrupt track switching, the website smoothly fades emotional layers in and out during scrolling.

The goal was to make the soundtrack feel subconscious rather than obvious.

---

## Philosophy

Modern websites optimize for speed.

This project optimizes for feeling.

It intentionally encourages:
- slower interaction
- emotional pacing
- atmosphere
- reflection

The experience is meant to be explored slowly.
